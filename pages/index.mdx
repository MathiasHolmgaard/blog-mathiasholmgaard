## Hvad ligger bag denne blog?
Dette er en markdown side, hostet med Vercel.com

Min fremgangsmåde er at skrive teksterne i Google docs.
Derigennem kan jeg kopiere teksten som Markdown format.

Det indsætter jeg i en .mdx fil under den passende mappe som i kan se ude til venstre.
Alle kategorierne er bare mapper den underliggende filstruktur.

Så comitter og pusher jeg mine rettelser til mit repository på github som så har actions som pusher opdateringen til blog.mathiasholmgaard.com
